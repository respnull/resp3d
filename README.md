# resp3d
A rotatable 3D model of a [respling](https://respnull.github.io/assets/thisisarespling.png) written in HTML5!<br>

## Link
You can access by [clicking here](https://respnull.github.io/resp3d).<br>

## Features
**F2 (Debug)** You can press **F2** to see most variables used by the project.<br>
**Rotating** Hold down your left mouse button and move left, right, up or down! Use your finger if mouse is not an option.<br>
**Zooming** Scroll down to zoom out, scroll up to zoom in! Use 2 fingers if mouse is not an option.<br>
**Disco** If you rotate the respling fast enough, he will change his expression, music will start and ***flashing lights will appear.***<br>
*You can disable Disco Mode by refreshing the page and selecting "Yes, i am sensitive." in the epilepsy warning.*<br>
